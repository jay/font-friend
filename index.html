This is a clone of Font Friend, a bookmarklet that can change the fonts on a page.

The original Font Friend stopped working in June 2016 after Google Code went into archive mode.

Drag the bookmarklet to your browser toolbar:

FontFriend

Note the @font-face drag-and-drop detection does not work.

Note the bookmarklet will likely not work on webpages that have a Content Security Policy, such as github.com webpages.

Also see davelab6/font-friend
